### What is this?

Personal [LazyVim](https://www.lazyvim.org/) configuration. Requires Neovim >= 0.11.3.

### Highlights

- **Colorscheme**: moonfly
- **Languages**: Elixir (Expert LSP), Go, Rust, TypeScript, Python, Docker, Terraform, SQL, and more (15 language extras)
- **Formatting**: conform.nvim (golines for Go, LSP for the rest)
- **Linting**: nvim-lint with per-language linters
- **Statusline**: search count, progress, location, active LSP server name
- **Pickers**: fullscreen Snacks picker with visible borders
- **Indent guides**: rainbow-colored, dimmed
- **Extras**: actions-preview, rainbow-delimiters, vim-better-whitespace, diagflow, scope, twilight
